About HouseWise

Intelligent Property Valuation System

How It Works

HouseWise uses a two-stage LLM prompt chain combined with a trained machine learning model. First, it extracts property features from your natural language description. Then, a Random Forest model (trained on 2,930 real Ames housing transactions) predicts the market value. Finally, an AI explains the price in plain English.

Stage 1: Extraction

LLM extracts 12 property features from your description. Never guesses - only uses what you mention.

Stage 2: Prediction

Random Forest model predicts price based on extracted features and training data.

Stage 3: Explanation

AI explains why the price is high or low, with key factors driving value.

Completeness Gate

Never predicts with missing data - asks for missing details first.
